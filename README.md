# 战网资源文件

## 地图要求
- 假如往地图内添加了脚本、队伍、命名单位，则它们的名称需要是英文，不能是中文
- 地图文件名必须是英文、数字和下划线，不能有额外的小数点、方括号、中文等字符。
  - 例子：`Bang_Guandu_1_6.map` 是可以的
  - 反面例子：`[Bang]Guandu_1.6.map` 是需要修改的
- 地图显示名称可以是任意字符，它们在 `map.str` 里被定义：
例如：
```
Map:Bang_Guandu_1_6
"Bang_Guandu[1.6]"
END

Map:Bang_Guandu_1_6?chinese
"三国杀之官渡之战[1.6]"
END
```
